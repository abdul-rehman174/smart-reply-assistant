# Kya Reply Karu?

Never stuck on what to text back. Paste the message you received, pick a vibe (smooth, funny, playing it cool, flirty, shut it down nicely), and get a perfect reply — in your own language and casual style. Single static HTML file; the AI runs on your own free Groq key.

## Features
- Reads the message and its subtext, then suggests replies that fit
- Replies in the same language and register you write in
- Multiple tones to choose from
- Installable PWA, auto-updates, works offline (the UI), no backend

## Run it
Open `index.html`, tap the gear, paste a free Groq API key from console.groq.com, and go. The key is stored only in your browser.
